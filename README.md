# App Business Boilerplate

앱 비즈니스에 필요한 기획, 디자인, 개발 산출물을 한곳에서 관리하는 보일러플레이트입니다.

## 폴더 구조

```
├── docs/           # 시장 조사, 경쟁 분석, 기획 문서
├── design/         # 디자인 에셋, 목업, 스크린샷
│   ├── mockups/
│   └── screenshots/
├── app/            # Flutter 앱 소스 코드
│   ├── docs/       # 앱 셋업 가이드, 기능별 사용법
│   ├── lib/        # Dart 소스 (Presentation / Domain / Data)
│   ├── test/       # 테스트
│   ├── android/
│   ├── ios/
│   └── README.md   # 앱 상세 안내
└── CLAUDE.md       # AI 코딩 규칙
```
