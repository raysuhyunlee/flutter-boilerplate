# 2. 리뷰 분석
```
[Agent 지시사항]

- 리뷰 원본은 같은 폴더에 csv 파일로 저장한다. csv 파일에는 앱 이름과 리뷰 별점, 리뷰 제목, 리뷰 내용, 작성자가 포함되어야 한다.
```

## 좋은 점
```
[Agent 지시사항]

- 표에는 빈도, 비율, 대표 리뷰 원문을 담는다.
```
## 개선할 점
```
[Agent 지시사항]
```

# 3. 키워드 분석
```
[Agent 지시사항]
- 해당 앱들과 가장 관련이 깊은 키워드 20~30개를 찾는다.
- astro MCP를 이용해 주요 키워드의 popularity와 difficulty를 분석한다.
- mcp-appstore MCP를 이용해 주요 키워드의 경쟁 앱 수와 평균 다운로드 수, 제목에 정확히 일치하는 키워드가 있는 앱 수를 분석한다.
- 분석 결과를 표로 정리한다.
  
주의: astro MCP를 쓸 때는 임시 앱을 생성해 해당 앱에 키워드를 추가한다. 기존 앱에 키워드를 추가하면 안 된다.
```
