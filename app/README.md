# App

Firebase Analytics, Crashlytics, DI(GetIt), 구독/페이월, 온보딩, 설정 등이 사전 구성된 Flutter 앱입니다.

> 모든 Flutter CLI 명령어(`flutter run`, `flutter test` 등)는 이 디렉토리에서 실행합니다.

## 제공 기능

| 기능 | 설명 | 문서 |
|------|------|------|
| **Firebase** | Analytics, Crashlytics 연동 | — |
| **DI** | GetIt 기반 의존성 주입 | — |
| **세션 관리** | 첫 실행 감지, 설치일 기록, 세션 카운트 | [docs/features/session-service.md](docs/features/session-service.md) |
| **온보딩** | 스와이프형 온보딩 페이지 (커스터마이징 가능) | [docs/features/onboarding.md](docs/features/onboarding.md) |
| **구독/페이월** | RevenueCat 기반 구독 관리 및 페이월 UI | [docs/features/subscription-paywall.md](docs/features/subscription-paywall.md) |
| **설정** | 리뷰, 공유, 약관/개인정보 링크, 확장 가능 | [docs/features/settings.md](docs/features/settings.md) |
| **Dev Tools** | 세션 조작, 초기화 등 개발자 도구 | [docs/features/dev-tools.md](docs/features/dev-tools.md) |
| **디자인 시스템** | AppColors, AppSizes, AppTextStyles, 테마 | — |
| **다국어** | 한/영 ARB 파일 기반 i18n | — |
| **공통 위젯** | CTA 버튼, 다이얼로그, 로딩, 스낵바 등 | — |
| **SQLite** | 로컬 데이터베이스 스캐폴딩 | — |
